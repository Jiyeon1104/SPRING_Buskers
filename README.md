✨프로젝트를 진행하면서 느낀 점<br>
Java Script를 제대로 사용하고 접해본 적이 없어서 처음엔 이벤트 먹이는 것조차 너무 많은 시행 착오를 거쳤다. Front에서는 Date picker API를 사용하는데 어려움이 있어 제대로 달력을 끝내 구현하지 못했지만 더 이상 지체할 수 있는 시간이 없다고 느껴 아쉽지만 select box로 변경하여 진행하였다. <br>
Back 작업에 들어가서는 약 2400줄에 육박하는 html을 접하여 막막하기도 하고 많은 근심과 걱정이 있었지만 타인의 코드를 요목조목 따져보고 분석해보는 시간이 이번 프로젝트의 가장 큰 수확인 것 같다. 처음에는 다른 팀원의 js코드를 제대로 활용하지 못하고 제대로 분석하지 못해 한 버튼에 클릭 이벤트가 2개씩 걸리게 코드를 작성하기도 했지만 팀원의 코드를 찬찬히 읽어보며 팀원의 js 코드를 적극 활용해보기로 마음먹어 이후에는 팀원이 작성한 js 이벤트를 토대로 rest작업을 진행하며 감을 잡았다.<br>
페이지 이동이 없게끔 하기 위해 ajax 코드를 약 15개 정도 작업하며 ajax에 대한 공부를 톡톡히 하게 된 것 같다.
